# Web App Template
Web app template for python frameworks(FastAPI, Flask, Bottle...)
Usage:
```sh
run.sh
```
### Default directory tree:
```
── app
    ├── database
    ├── job
    ├── logs
    ├── mail
    ├── src
    │   ├── certificates
    │   │   └── inst.crt
    │   ├── docker-compose.yml
    │   ├── Dockerfile
    │   ├── main.py
    │   ├── requirements.txt
    │   ├── run.sh
    │   ├── settings.py
    │   ├── static
    │   │   └── bulma.css
    │   └── templates
    │       ├── auth
    │       └── base.html
    └── upload
```
